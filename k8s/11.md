### Create a secret
$ kubectl create secret generic my-secret --from-literal=username='admin' --from-literal=password='admin1234'
secret/my-secret created

### Verify and decode
kubectl describe secret my-secret
Name:         my-secret
Namespace:    default
Labels:       <none>
Annotations:  <none>

Type:  Opaque

Data
====

username:  5 bytes
